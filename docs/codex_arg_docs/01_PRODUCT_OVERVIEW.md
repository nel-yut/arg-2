# プロダクト概要

## 目的
本作は、ブログから始まり、宗教団体の公開サイト、資料室、内部記録、出航記録へと進んでいく検索型 ARG である。

## フェーズ構成
- **Phase0**: ブログ導入
- **Phase1**: 公開サイト探索 / 資料室発見
- **Phase2**: 資料室探索 / 第3教育寮 / 反応矯正教育の不穏さ
- **Phase3**: 反応矯正教育の深掘り / 第6教育寮 / 記録の照合 / 渡航巡礼への接続
- **Phase4**: 渡航巡礼の実態解体 / 出航記録 / 死亡根拠
- **Phase5**: Ending
- **Phase-End**: ブログ404や終端用ページ

## 体験のコア
- プレイヤーはリンクではなく **検索で先へ進む**
- 検索できる範囲は Phase によって制限される
- 1つの検索ワードから複数件返ることがある
- しかし検索は完全一致なので、プレイヤーが拾った語をそのまま打つ必要がある

## ブログと本体サイト
### ブログ
- ドメイン: `ino-blog.nel-app`
- 本体と別レイアウト
- 失踪した友人を探す個人ブログ
- 本体サイトへの直接リンクを含む

### 本体サイト
- 宗教団体サイト + 資料室
- 右上に検索UI
- 検索結果一覧からページへ進む
- ページ右下に `PhaseX (n)` を表示

## プレイヤーに見せない内部状態
- `currentPhase`
- `readPages`

## プレイヤーに見せるもの
- ページ本文
- タイトル
- 検索結果タイトル一覧
- 画像（必要なページのみ）
- 右下の Phase 表示
