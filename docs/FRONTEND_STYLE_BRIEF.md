# Frontend Style Brief

## 目的
このARGサイトのフロントエンドは、機能実装だけでなく「実在する古いサイトとして自然に見えること」が重要である。
ブログと本体サイトは別物として見せるが、どちらも読める・探せる・壊れて見えないことを優先する。

## 全体方針
- 派手な演出は不要
- ホラー演出よりも「実在感」「古さ」「素朴さ」を優先する
- UI は静的でよいが、余白と整列は丁寧に行う
- 情報が多いページでも、読みづらくしてはいけない
- 実装者の好みで現代的なダッシュボード風デザインにしない
- ガラス風、グラデーション、強い影、派手なアニメーションは禁止

## デザインの基調
### 本体サイト
- 古い宗教団体・教育団体の公式サイトのような見た目
- 白背景またはごく薄い生成り背景
- 濃いグレー本文
- リンクは古いWebらしい青系でもよい
- 横幅は広げすぎず、本文は読みやすい最大幅を設定する
- ヘッダー、本文、フッターの3層構造を明確にする
- 右上に検索UIを置く
- 右下に小さく Phase とページ番号を表示する

### ブログ
- 本体サイトとは完全に別レイアウト
- 個人ブログらしい素朴さを優先
- 少し雑多でもよいが、崩れてはいけない
- 記事一覧、本文、サイドバーがあってもよい
- 検索UIは置かない

## レイアウト要件
- モバイルよりもPC表示を主対象とするが、最低限レスポンシブにする
- 画面左右に十分な余白をとる
- 本文は長文を読む前提で行間を狭くしすぎない
- タイトル、本文、補足情報の階層を視覚的に分ける
- ページ上部のタイトルは十分目立たせる
- セクション間の余白は明確にとる
- テキストが画面端に張り付く状態は禁止
- 検索結果一覧はタイトルだけを縦に並べ、余計な装飾は避ける

## タイポグラフィ
- 日本語長文が読みやすい設定を優先する
- 本文サイズは小さすぎない
- 行間は十分に確保する
- タイトル、見出し、本文、注記のサイズ差を明確にする
- 右下の Phase 表示は小さく、しかし読めるサイズを保つ

## 共通レイアウト仕様

### 福音継承教育会HP・共通項目
本体サイトの通常ページは、一般的な宗教団体HPとして見える共通レイアウトを持つ。
サイト名は「福音継承教育会」とする。
ページ右上には検索ボックスを配置し、検索後は検索結果一覧ページへ遷移する。
ページ上部には、マウスオーバーで開くグローバルメニューを配置する。
メニュー項目は以下に限定する。

- TOPページ
- 福音継承教育会とは
- 定例行事のご案内
- 施設について
- 入会のご案内
- よくある質問

### 資料室・共通項目
資料室ページは、本体サイト通常ページとは別の共通レイアウトを持つ。
秘匿資料ページのような印象を与えるため、背景色は黒とする。
ページ右上には検索ボックスを配置し、検索後はHP側の検索結果一覧ページへ遷移する。
ただし、資料室ページにはグローバルメニューを設置しない。
資料室内の他ページへの移動手段は検索ボックスのみとする。

## コンポーネント要件
### 検索UI
- 右上固定ではなく、ページ上部右寄せ配置でよい
- 1入力欄 + 1ボタンのシンプル構成
- 複数ワード検索不可の仕様を壊さない
- 見た目は簡素でよいが、ボタン・入力欄の高さと余白は揃える

### 検索結果
- タイトルのみ表示
- クリック可能であることが明確にわかる
- 過剰なカードUIにしない
- 結果0件時の注意文は読みやすく表示する

### 本文エリア
- テキスト主体
- 画像があるページでも本文の流れを壊さない
- 黒塗り画像や記録画像は、本文よりやや狭い横幅で見せてもよい
- 画像キャプションを付けられる構造にしておくと望ましい

## 画像表示
- 画像はページ幅いっぱいにしない
- 画像周囲に余白をとる
- 画像が複数ある場合は縦積みを基本にする
- 画像は「資料」「記録」「写真」のように自然に差し込む
- 画像の角丸や派手な枠線は不要
- 古いサイトらしく、簡素な枠線程度でよい

## 実装上の禁止事項
- 勝手に現代アプリ風のUIへ寄せない
- コンテンツの文言を見た目都合で省略しない
- 機能を増やさない
- フェーズ制御や検索仕様を変えない
- 過剰なアニメーションを付けない
- Tailwind の派手な色や装飾を乱用しない

## 完了条件
以下を満たしたら「見た目が整った」と判断する。

1. 各ページで余白・行間・文字サイズが破綻していない
2. 本体サイトとブログの見た目が明確に別物である
3. 検索UIが目立ちすぎず、しかし見つけやすい
4. 検索結果一覧がシンプルで見やすい
5. 長文ページが読みにくくない
6. 画像を置いたときに崩れない
7. 右下の Phase 表示が邪魔にならず確認できる
8. 古いサイトの実在感があり、安っぽいモダンUIになっていない
