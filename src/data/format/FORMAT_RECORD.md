# 「記録：」用フォーマット
## ページコンテンツフォーマット
- 各コンテンツページのフォーマットは統一感を持たせるため以下のフォーマットを採用する。
- `{}`で囲まれた箇所は変数である
- また、マスクされている箇所は`■`で表す(文字数分使用する)
- 口調をである調で統一する
- ページ全体の文量を800-1200文字程度とする

## フォーマット
個体識別子：{期}-{3桁連番}
個体呼称：{氏名フルネーム}({氏名フルネームかな})
性別：{男or女}
生年月日：{西暦}年 {月}月 {日}日
入信日：{西暦}年 {月}月 {日}日

{本人についての備考}

{第6教育寮課程についての記載}

{実験についての結果。}

{洗脳についての結果。性格について}
