# 「出航記録」用フォーマット
## ページコンテンツフォーマット
- 各コンテンツページのフォーマットは統一感を持たせるため以下のフォーマットを採用する。
- `{}`で囲まれた箇所は変数である
- また、マスクされている箇所は`■`で表す(文字数分使用する)
- 口調をである調で統一する
- ページ全体の文量を1000-1500文字程度とする

## フォーマット
出航日程：{西暦}年 {月}月 {日}日
出航先：{国名}
出航人数：{人数} 名

当該工程においては{出航した人物名の列挙(句点区切り)}の{人数}名が第{n}期対象として渡航巡礼した記録である。

{今回の渡航巡礼の概要}

{個体1についての記録。選定理由、船内記録、受け渡し時の記録}

{個体2についての記録。選定理由、船内記録、受け渡し時の記録}

{個体の数分、当該個体の記録を記載。選定理由、船内記録、受け渡し時の記録}

{渡航先での個体の処遇について}

{個体についての雑記}

{渡航について雑記}

{記録者の所感}
